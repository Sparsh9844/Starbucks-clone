# ☕ Starbucks Website Clone

A responsive Starbucks website clone built using **HTML**, **CSS**, and **Bootstrap**.  
This project is created for learning and practicing frontend web development skills.

---

## 🚀 Features

- Responsive design
- Starbucks-inspired modern UI
- Bootstrap layout and components
- Product categories section
- Carousel section
- Navbar and footer
- Mobile-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
